GUIDE

Options available on the chart

  • User can use the mouse wheel to zoom-in & zoom-out of the chart or use zoom with trackpad as well
  • Once zoomed, user can click and drag on the chart to pan around the chart
  • User can drag the point to change its value
  • When clicking on a point, a new popup will be available, where the user can type any specific value to change the point.
  • On the popup, user can use keyboard's 'up' and 'down' arrows to change the values, when holding shift, and the value will change with an increment of 10
  • Hovering over a point will reveal more information about the point in a tooltip, also line along the yAxis with annotation about the date
